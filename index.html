One command installs token-saving skills into every AI assistant you use — Claude Code, Copilot, Cursor, Windsurf, Aider — simultaneously, per project.

npx tokenimizer init
How it works
01
Skill files
Instruction blocks appended to each assistant's config. Read on every session start. Zero runtime overhead.
02
Git hooks
Context docs refresh on commit. Index rebuilds on merge. Stale warnings on push. Never blocks a git operation.
03
Background watcher
Optional file-system daemon. Marks the repo index stale when source files change. Writes only to .tokenimizer/.
